Мой стек технологий

    Язык C#

  • Современный высокоуровневый язык
  • Поддержка как ООП, так и ФП парадигм
  • Уровень владения: Продвинутый

    Платформа WPF

  • Современная система для построения клиентских приложений
  • Обеспечивает визуально привлекательные возможности взаимодействия с пользователем
  • Уровень владения: Продвинутый

    Паттерн MVVM

  • Паттерн MVVM - современный подход к организации структуры приложений, создаваемых на платформе WPF
  • Вносит в структуру приложения гибкость и возможность модификаций
  • Уровень владения: Мастер

    СУБД PostgreSQL

  • Современная мощная система управления базами данных
  • Самая сбалансированная и популярная реляционная СУБД
  • Уровень владения: Продвинутый

    Docker

  • Docker - контейнеризатор приложений
  • Самое выгодное программное решение для контейнеризации
  • Уровень владения: Начальный

    Язык разметки HTML

  • HTML - современный язык разметки, разработанный специально для создания структуры веб-сайтов
  • Позволяет создавать гибкую и отзывчивую структуру сайтов
  • Уровень владения: Средний

    CSS

  • CSS - формальный язык описания внешнего вида веб-страницы
  • Предоставляет удобный инструментарий для оформления стилей веб-страницы
  • Уровень владения: Средний

    Язык JavaScript

  • JavaScript - главный язык программирования, работающий на уровне веб-сайта
  • Вносит дополнительные структурные компоненты, обеспечивая богатое разнообразие выразительных средств для веб-страницы
  • Уровень владения: Начальный
